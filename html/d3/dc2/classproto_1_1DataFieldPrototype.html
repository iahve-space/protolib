Detailed Description

template<typename PACKETS, uint8_t * BASE, FieldFlags FLAGS, size_t MAX_SIZE = 4096>
class proto::DataFieldPrototype< PACKETS, BASE, FLAGS, MAX_SIZE >

Field prototype representing a DATA field that can contain one of multiple packet types selected by ID.

Template Parameters
PACKETSstd::tuple of PacketInfo entries describing available payloads.
BASEBase pointer to shared buffer.
FLAGSField flags.
MAX_SIZEMaximum size in bytes (applies to pointer payloads).
