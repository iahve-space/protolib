Detailed Description

Helpers and utilities for compile-time field metadata and tuple processing.

This header provides:

  • proto::FieldInfo — a lightweight metadata wrapper for a field pointer and its size;
  • proto::MakeFieldInfo — a helper to construct FieldInfo with deduction;
  • proto::FieldInfoHasName — compile-time check for presence of a field by name in a tuple;
  • proto::GetFieldInfoByName — compile-time retrieval of a field info by name from a tuple;
  • proto::ForEachInfo — compile-time for-each over a tuple;
  • proto::AppendToInfo — helper to append a new element to a tuple at compile time.
