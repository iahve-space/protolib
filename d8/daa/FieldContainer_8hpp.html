Detailed Description

Defines the FieldContainer class template, a generic container for protocol fields.

This header provides a generic container template that holds protocol field instances, manages CRC calculation, supports debug facilities, and provides convenient access and iteration over the fields by name or index. The container is designed to work with protocol field types, enabling lookup by name or index, state reset, and debug output for protocol development and diagnostics.
