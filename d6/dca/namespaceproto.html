◆ MatcherType

using proto::MatcherType = typedef MatchStatus (*)(void*)

Maximum allowed field size in bytes.

Matcher callback type.

A matcher inspects the current field buffer and returns MatchStatus. Signature uses a raw pointer to the field storage; implementers should cast to the appropriate type for read-only checks. (The pointer is not owned and must not be stored.)
